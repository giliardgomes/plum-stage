## Typography

### Font Family
```typescript
fontFamily: "Roboto, Arial, sans-serif"
```

### Font Sizes
```typescript
fontSizes: {
    xs: "0.75rem",    // 12px
    sm: "0.875rem",   // 14px
    md: "1rem",       // 16px
    lg: "1.125rem",   // 18px
    xl: "1.375rem"    // 22px
}
```

### Headings
- Font Weight: 500
- XS Heading:
  - Font Size: 18px
  - Line Height: 24px
  - Letter Spacing: 0.3px

### Paragraph
- Medium:
  - Font Size: 14px
  - Line Height: 20px
  - Letter Spacing: 0.3px

## Spacing
```typescript
spacing: {
    xs: "0.5rem",     // 8px
    sm: "0.75rem",    // 12px
    md: "1rem",       // 16px
    lg: "1.25rem",    // 20px
    xl: "1.5rem"      // 24px
}
```

## Border Radius
Default radius: 0.25rem (4px)
